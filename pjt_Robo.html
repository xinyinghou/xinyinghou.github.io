Description

These is my summer school project about students' math game performance in RoboTutor.

We uncovered the potential learning difficulties that K-2 students faced while learning about missing number completion in a tablet-based program called RoboTutor. We analyzed 185,064 rows of log-data from 283 students to determine the tutor pass rates and made learning curves for each activity within that knowledge component.
